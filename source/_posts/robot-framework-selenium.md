## 安装brew ##

```sh
curl -LsSf http://github.com/mxcl/homebrew/tarball/master | sudo tar xvz -C/usr/local --strip 1
```

## 安装wxpython ##

```sh
brew install wxpython
brew info wxpython
```

## 安装phantomJS ##

```sh
pip install selenium
```
